## 개발 환경
- npm 6.14.15
- react 17.0.2
- next 12.1.2

<br />

## 사용 방법
### 코드 포맷팅
```sh
command + alt + shift + p
```

### 라이브러리 설치
```sh
$ npm install
```

### developer 환경으로 실행
```sh
$ npm run dev
```
